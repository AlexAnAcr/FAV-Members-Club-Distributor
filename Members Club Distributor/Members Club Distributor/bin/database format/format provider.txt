----ALL FORMAT----
Файл "info.ini", находящийся в коренной папке базы данных, во всех форматах базы данных содержит следующие строки:
----
[format]
%f%

[versions]
%v1%
%v2%
%v3%
<...>
----
Пустые строки в файле не имеют значения и могут отсутствовать, или наоборот - присутствовать в обилии, содержимое раздела "format" и "versions" должно быть считано обязательно.
%f% - это формат базы данных
%v1%, %v2%, %v3% - это список версий программ, работавших с базой данных, метка <...> означает "и т.д." (%v4%, %v5%)
%v1% - версия программы, создавшая базу данных.
<...> - версия программы, работавшая с базой данных последний раз.

Внимание: файл "info.ini" сам версий не имеет и ведётся во всех версиях MCD одинаково!


----LmsI-CLrw-6o2m1i----

LmsI-CLrw-6o2m1i - Формат базы MCD, хранящий данные в 2 файлах: файле данных и изображении. В "коллекторе" указан только файл данных. Совместимые версии MCD: 2.0
1  2 34   4

1 - Line main storage - файл данных
2 - Image - картинка
3 - Информация о коллекторе
4 - Last read write - коллектор содержит данные о последнем чтении и записи базы данных.
5 - 6o2m1i - 6 one line, 2 multi line, 1 image - формат файла данных

В этом формате пустая строка считается за конец раздела, объявление нового раздела за конец предыдущего не считается, конец файла считается за конец раздела.

Пример 1:
--------
[config]
param1=1

param2=2
--------
param2 не будет обнаружен  в "config", т.к. раздел "config" завершён.

Пример 2:
--------
[config 1]
param1=1
[config 2]
param2=2
--------
Здесь произойдёт ошибка, т.к. "config 1" не был завершён к началу "config 2".
